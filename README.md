En este trabajo, se busca implementar un algoritmo genético que ayude a la generación de tableros de datos, más específicamente en el posicionamiento y tamaño de las gráficas deseadas. Los tableros de datos sirven como estructuras que almacenan diferentes representaciones gráficas de información. Por ello, en este caso, dado un determinado número de gráficos, se establece una serie de rectángulos donde se pueden colocar las gráficas una vez que se cuente con ellas, proponiendo una interfaz intuitiva y sencilla de utilizar.


Hoy en día los tableros de datos, también conocidos como dashboards, se han vuelto una herramienta cada vez más común, debido a su que resultan bastante prácticos para la representación directa y gráfica de grandes cantidades de información, que permite sea más fácil de interpretar y distribuir.

Estos mezclan el conocimiento con la estética, por lo que en diversas ocasiones se pueden generar complicaciones al momento de establecer las posiciones de los gráficos, asegurando que se mantenga la distribución del espacio general, así como el tamaño suficiente para colocar un gráfico legible.

Aquí es donde entra el uso de los algoritmos genéticos, ya que resulta un problema de optimización que puede resolverse mediante la búsqueda de soluciones, por lo que se aprovecharán para la creación de este trabajo, y buscar la forma en que se generen las posiciones deseadas y con la mejor distribución posible.
